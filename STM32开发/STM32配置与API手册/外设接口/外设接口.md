### LL库 无DMA

代码段 `MX_ADCx_Init` 中增加：
```C
LL_ADC_Enable(ADCx);                       // 使能ADC 
LL_ADC_StartCalibration(ADCx);             // 校准ADC
while(LL_ADC_IsCalibrationOnGoing(ADCx));  // 等待ADC校准完成
```
